# Data Structures and Algorithms

This repository contains solutions to various Data Structures and Algorithms (DSA) problems implemented in Java.

## Content Overview

The repository includes multiple Java files focusing on different algorithmic domains, ranging from simple array manipulations to advanced dynamic programming and graph problems.

### Problem Categories Included:
- **Dynamic Programming & Recursion:** Burst Balloons (`burst_balloon.java`), N-Queens (`nqueens.java`), Frog Jump (`frogjump.java`), Stair Climbing (`stair.java`), Palindrome Partitioning (`palindrome_partition2.java`), House Robber (`rob.java`), Subsets (`subset1.java`)
- **Linked Lists:** Cycle Detection (`detect_cycle.java`, `ll_cycle.java`), Adding Lists (`add_list.java`), Removing Elements (`remove.java`)
- **Strings:** Longest Palindromic String (`lps.java`), Substring Counting (`count_binary_substring.java`), Word Reversal (`reverse_word_string.java`), Minimum Deletions (`minimum_deletion_string.java`)
- **Graphs / BFS / DFS:** Distinct Islands (`distinct_islands.java`), Word Ladder (`word_ladder1.java`)
- **Arrays & Mathematics:** Sum of Subarray Minimums (`subarray_sumMin.java`), Finding Unique Elements (`unique.java`, `unique2.java`), Finding Duplicates (`duplicate.java`), Max Element (`max.java`)
- **Bit Manipulation:** Decimal to Binary (`dtob.java`), Minimum Bit Flips (`min_flip_bits.java`)

## Getting Started

To compile and run any of the solutions, you need to have the Java Development Kit (JDK) installed. 

### Running a Solution

1. Open your terminal/command prompt.
2. Navigate to this directory.
3. Compile the desired Java file:
   ```bash
   javac filename.java
   ```
4. Run the compiled Java class:
   ```bash
   java filename
   ```
   *(Ensure to replace `filename` with the actual file/class name without the `.class` extension)*

## Prerequisites
- [Java Development Kit (JDK)](https://www.oracle.com/java/technologies/downloads/) 8 or higher.
